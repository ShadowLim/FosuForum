# FosuForum
佛大校园论坛

本项目采用基于BootStrap + Jsp + Servlet + MySQL+ Tomcat的开发模式和Maven构建工具进行佛大校园论坛的制作。 

实现的功能包括：用户（或管理员）登录，发表新的帖子，查看帖子，回复帖子（也叫跟帖）以及管理员删除发言不当帖子或着跟帖，首页轮播图，分页查询和模糊搜索功能，管理员实现用户管理，数据报表展示，以及佛大周边交通模块。
